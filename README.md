# Manufacturing Production Analysis
## Tools Used
- Power BI
- Dataset: Predicting Manufacturing Defects (Kaggle)

## Dashboards
- Dashboard 1: Production & Efficiency Overview
- Dashboard 2: Quality & Cost Analysis

## Key Insights
1. Delivery delay increases defect rate
2. 84% products are High Defect
3. Low Defect products have higher quality scores
4. Higher supplier quality = fewer defects
